Editing tips (add / delete cards)
1) Scroll to the bottom of this file and find CARD DATA.
2) Each card is one JSON object with an id. To delete a card, remove that whole object.
3) To add a card, copy/paste the template object and fill it in.
Tip: keep the commas between objects. Last object should NOT end with a trailing comma.
Tip: click a card to open a large viewer (front/back).
